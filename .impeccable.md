## Design Context

### Users
Solo creator managing multiple short-video channels across TikTok, YouTube Shorts, Instagram Reels, and Facebook Reels. Power user who monitors dozens of channels simultaneously — needs dense information display without cognitive overload. Primary context: desktop browser, often in low-light environment (late-night content scheduling). Later: multi-tenant SaaS for other creators.

### Brand Personality
Precise, relentless, intelligent. Think "mission control for viral content" — the interface should feel like a well-calibrated instrument panel, not a marketing landing page. Every pixel earns its place.

### Aesthetic Direction
- **Visual tone**: Linear-inspired dark-mode-native. Near-black canvas (#08090a), content emerges through luminance hierarchy. Ultra-minimal chrome, maximum signal.
- **Reference**: Linear (primary), Vercel (secondary for data visualization patterns)
- **Anti-references**: Colorful SaaS dashboards (Notion rainbow), generic shadcn/ui dark themes with purple gradients, any "AI-generated" aesthetic with glassmorphism/neon accents
- **Theme**: Dark mode only (Phase 1). Light mode is explicitly out of scope.
- **Design system**: See `DESIGN.md` (Linear-inspired, installed via getdesign.md)

### Design Principles
1. **Density over decoration** — Show more data in less space. No decorative illustrations, hero images, or empty state artwork. Information IS the interface.
2. **Darkness as medium** — The near-black background is not a "dark theme applied to a light design." It's the native canvas. Depth comes from luminance stepping, not shadows.
3. **One accent, one purpose** — Brand indigo (#5e6ad2 / #7170ff) is reserved exclusively for primary CTAs and interactive accents. Everything else is grayscale.
4. **Precision typography** — Inter Variable with cv01+ss03, weight 510 as the workhorse. Aggressive negative letter-spacing at display sizes. This is non-negotiable.
5. **No AI slop** — Zero tolerance for gradient text, side-stripe borders, nested cards, glassmorphism, bounce animations, or the cyan-on-dark "AI color palette." If it looks like an AI built it, redesign it.
